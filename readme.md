# blog

Implementación básica de DtaTable en Laravel utilizando el paquete yajra/laravel-datatables.

## Install

<p align="center">

$ composer install

</p>

<p align="center">

$  cp .env.example .env

</p>

<p align="center">

$ php artisan key:generate

</p>

<p align="center">

$ php artisan migrate

</p>

<p align="center">

$ php artisan db:seed

</p>
